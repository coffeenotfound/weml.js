Vec4

new Vec4(xopt, yopt, zopt, wopt) → {Vec4}

Constructs a new Vec4 with the given values. The components of the vector are ordered as (x, y, z, w). The given values default to the identity of a Vec4 which is (0, 0, 0, 1).
Parameters:
Name Type Attributes Default Description
x Number <optional>
 0 The initial x component of the vector
y Number <optional>
 0 The initial y component of the vector
z Number <optional>
 0 The initial z component of the vector
w Number <optional>
 1 The initial w component of the vector
